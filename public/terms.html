The short version

Play games for free. Don't abuse the platform. We're one person running this as a hobby.

Use of the platform

Hollowed Stone provides free online board games. You may use the platform to play games, view rulebooks, and submit feedback. No account is required.

No accounts, no personal data collection

We do not create user accounts. We do not collect names, emails, or passwords during gameplay. The only data stored is:

  • Game state (board positions, moves) stored temporarily for active games
  • IP addresses logged for rate limiting and abuse prevention
  • Feedback submissions (only if you voluntarily submit the feedback form)

Game data

Active games expire after 7 days of inactivity. Finished games are retained for 30 days for replay purposes, then automatically deleted. Access codes are recycled after expiration.

Acceptable use

Do not:

  • Flood the API with automated requests beyond normal gameplay
  • Attempt to interfere with other players' games
  • Submit false or abusive feedback
  • Use the platform for any illegal purpose

Open source

The source code is available on GitHub under the MIT license. You are free to fork, modify, and host your own instance.

The platform is provided as-is, with no guarantees of uptime, availability, or data preservation. Games may be lost due to server issues. This is a hobby project, not a commercial service.
